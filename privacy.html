RavenGuard Privacy Policy

Effective Date: 04/06/2026

RavenGuard analyzes visible Gmail email content in the browser to flag suspicious messages. It does not collect or transmit data externally.

Trusted Contact Feature

If the user chooses, RavenGuard can open a Gmail draft to a trusted contact containing details from the flagged email for review.

Local Storage

Trusted contact details and user name are stored locally using Chrome storage and are never transmitted externally.

Permissions

  • Access to Gmail pages for analysis
  • Storage for saving user preferences
